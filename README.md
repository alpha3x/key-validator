This package is a simple key validator for php.

## Installation

You can install the package via composer:

```bash
composer require alpha3x/key-validator
```

## Usage

```php
$skeleton = new Alpha3x\KeyValidator();
echo $skeleton->echoPhrase('Hello, Alpha3x!');
```

## Testing

```bash
composer test
```
